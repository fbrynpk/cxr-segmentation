# Chest-XRay-Segmentation

Chest X-ray segmentation using U-Net models trained for anatomical structure segmentation.

## Models

* **Lung Segmentation** – Binary U-Net model for segmenting the left and right lungs as a single foreground class.
* **Heart Segmentation** – Binary U-Net model for segmenting the heart.
* **Lung + Heart Segmentation** – Multiclass U-Net model for segmenting the lungs and heart as separate classes.

## Features

* PyTorch implementation of U-Net
* Trained models for binary and multiclass segmentation
* Inference and evaluation scripts
